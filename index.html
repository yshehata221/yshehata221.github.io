AI-Generated Phishing: Detection & Mitigation

Can machine learning still catch phishing when the phishing is written by machines?

Phishing used to be easy to spot. Bad grammar, odd phrasing, the usual tells. LLMs fixed all of that for the attackers. I wanted to know whether a classifier could still separate machine-written phishing from legitimate mail, so I generated synthetic phishing with GPT, Claude, Gemini, Grok and DeepSeek across multiple prompt styles, mixed it into a dataset of real emails, and trained a Bidirectional LSTM to sort malicious from safe. The project covers both sides of the problem: how attackers can abuse LLMs, and which detection and mitigation approaches hold up against machine-written phishing compared with the human-written kind.
